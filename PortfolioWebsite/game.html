Subject Description

The lecture topics will cover essential knowledge for students to advance in game development using modern technology. These topics may include:
• Design patterns,
• Graphics,
• Physics in games,
• Character development,
• Animation,
• Game-specific AI,
• Implementing game mechanics,
• Game production process,
• Project management in teams,
• Memory management,
• Algorithm efficiency,
• Advanced programming techniques.

What I Have Learned

I learned a lot about animation, event-driven function handling, control mechanisms, and UI design to enhance the user experience in video games. Most of what I learned is highly specific to game programming, and even more narrowly focused on using Unity. As a result, transitioning to another game engine would likely be challenging

The game Idea

Our idea was to create a running game where the player spawns and tries to reach the finish line in various levels. The unique twist is that the player can throw a ball to teleport to its location. Beyond that, we experimented with different ideas, controls, and overall game feel during development, adding features like camera movements, camera shake, sound effects, and particle effects.
